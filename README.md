# Web Aggregator

A content aggregation platform for collecting and displaying web content.

## 🎯 Features

- RSS/Atom feed aggregation
- Web scraping
- Content categorization
- Search and filtering
- Automated content fetching
- Custom feed sources
- Content preview
- Bookmark management

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/none-ai/web-aggregator.git
cd web-aggregator

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
```

Visit http://localhost:5000

## 📋 Requirements

- Python 3.8+
- Flask
- Feedparser
- BeautifulSoup
- SQLAlchemy
- See `requirements.txt` for full list
